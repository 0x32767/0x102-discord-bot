0x102

intro

0x102 is a general purpose discord bot. The bot has many features such as: - custom commands - api commands - leveling - custom poles - mine craft - moderation - trivia quiz

Owner

The owner of the bot is @0x32767 and the bot can also be found on their github repository. The bot is also part of a chain of discord bots created by the DBC (discord bot corporation). All bots that have been made by the DBC are and will still be made by their owners. All that the DBC does is connect the bots together to a database to improve the user's experience.

Custom Commands

Custom commands are written in a programing language called lua. The developer chose lua because it is a simple language that is easy to learn. it is also roblox's language of choice when it comes to creating games. These commands are written to a file called commands.lua. The commands are public by default, however the commands will need to meet a strict standard to be made public.
To make a command you start by using /newcommand a modal wil popup which will prompt you to enter the program for the command. When you are finished you can submit the code and click the save button on a message sent to you. This will send the command to a database where it will be stored. These commands can be used publicly immediately.
Commands can also be part of the bot if a github pul request is sent. The command would then be part of the bot and can be used in any server (without having to reload the page or anything else).

Command standards

custom commands can currently not use the require function. This is because if it could the bot would be unsecure. The command context is also very restricted. The command only gives access to be able to respond to messages in plain text, and get the name of the command user.

Future support

The developer will start implementing features into the api, e.g more command context (less restrictions) and embed and modal support, as well as ui and api support.
